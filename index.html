Middlewares

Voor het schrijven van de log op de SD card wordt gebruik gemaakt van een stukje code in Middlewares/gwtonn/sd_logger.c. Deze code is betrekkelijk simpel. In het vervolg moet hier de logica zitten om de juiste info in de file te krijgen.

[!IMPORTANT] Voor de startLogTask is 3000 WORDS gereserveerd op de stack. We moeten dus opletten met de code (en dus ook wat opschonen).

De functie startLogTask wacht op een item in de queue. Wanneer deze binnen komt, dan wordt er een string gemaakt. Deze string wordt naar de UART gestuurd en ook naar de SD_LOGGER. Voor de sd_logger wordt ook gebruik gemaakt van een MUTEX om te voorkomen dat er 2 schrijfacties tegelijk plaatsvinden.

Voor mee details kan je kijken in de beschrijving van de bibliotheek.
